# 菜品图片目录

请将菜品图片放置在此目录下。

## 图片命名规范

图片命名建议使用菜品名称的拼音，如：

- `gongbaojiding.jpg` - 宫保鸡丁
- `fanqiejidan.jpg` - 番茄鸡蛋
- `hongshaorou.jpg` - 红烧肉
- `mapodoufu.jpg` - 麻婆豆腐
- `tangculiji.jpg` - 糖醋里脊
- `qingzhengluyu.jpg` - 清蒸鲈鱼
- `yuxiangrousi.jpg` - 鱼香肉丝
- `shuizhuyu.jpg` - 水煮鱼
- `huiguorou.jpg` - 回锅肉
- `baiqieji.jpg` - 白切鸡
- `suanrongfensizhengshanbei.jpg` - 蒜蓉粉丝蒸扇贝
- `jingjiangrousi.jpg` - 京酱肉丝
- `suancaiyu.jpg` - 酸菜鱼
- `ganbiandoujiao.jpg` - 干煸豆角
- `baizhuoxia.jpg` - 白灼虾
- `koushuiji.jpg` - 口水鸡
- `tangcupaigu.jpg` - 糖醋排骨
- `mayishangshu.jpg` - 蚂蚁上树
- `jiaoyanpaigu.jpg` - 椒盐排骨
- `yuxiangqiezi.jpg` - 鱼香茄子
- `default.jpg` - 默认图片（当菜品图片不存在时使用）

## 图片格式

支持 JPG、PNG 等常见图片格式。

## 访问路径

图片访问路径为：`http://localhost:8080/api/images/图片文件名`

例如：`http://localhost:8080/api/images/gongbaojiding.jpg`
